Facebook Canvas


The Climate Change Action Plan campaign reached audiences by using the isometric visual style on a Facebook canvas. The canvas is a relatively new tool that allows audiences to visit a longer format post within Facebook. The canvas reached 695,295 people on Facebook, received 1,303 total reactions of which 1,041 likes and 210 shares and received 8,664 clicks to learn more about the CCAP.
